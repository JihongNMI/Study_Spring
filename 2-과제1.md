# 구조

```
클라이언트 (POST 요청)
        ↓
Controller
  ↕ (DTO) ← 클라이언트 입력/출력
        ↓
Service
  ↕ (DTO/VO) ← 내부 로직 처리
        ↓
MyBatis Mapper
  ↕ (VO) ← DB 레코드 매핑
        ↓
DataSource (HikariCP = DBConPool)
        ↓
JDBC
        ↓
JDBC Driver
        ↓
DB
```

# MyBatis/Mapper 구조에서는 DAO가 안 보이는 이유

MyBatis에서 Mapper가 DAO 역할을 대신함

Mapper 인터페이스 + XML/Annotation = DAO와 동일 기능

SQL 실행 + 결과 매핑까지 처리
